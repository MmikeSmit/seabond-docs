# 欢迎使用 SeaBond

SeaBond 是面向 Hyperliquid 用户的聪明钱发现、钱包分析与复制交易平台。它帮助用户从公开交易数据中筛选高质量钱包，分析交易行为，并在理解风险后创建复制交易策略。

SeaBond 的核心价值不是“告诉用户买什么”，而是把钱包筛选、交易复盘、复制交易管理和账户监控集中到一个流程里，让用户能更系统地完成从发现到跟踪的决策。

## SeaBond 可以做什么

| 模块 | 作用 |
| --- | --- |
| 聪明钱发现 | 按账户规模、收益、胜率、交易频率、持仓方向等条件筛选 Hyperliquid 钱包。 |
| 钱包详情分析 | 查看目标钱包的账户权益、PnL、持仓、成交历史、资金流和交易风格。 |
| 复制交易 | 选择目标钱包后，创建复制交易规则，并在独立的复制交易账户中管理资金。 |
| 个人中心 | 查看总资产、复制交易权益、主钱包统一交易账户和个人资料。 |
| 收藏与标签 | 将观察中的钱包加入收藏，并用标签记录判断和分组。 |
| 邀请与积分 | 查看邀请码、邀请关系、奖励状态和积分记录。 |

## 建议使用流程

1. 先完成登录并连接钱包。
2. 在聪明钱发现页筛选候选钱包。
3. 进入钱包详情页复盘交易行为。
4. 将合适的钱包加入收藏并添加标签。
5. 如果决定跟随，再创建复制交易。
6. 在个人中心持续查看账户、复制交易权益和历史记录。

{% hint style="warning" %}
SeaBond 提供数据分析和交易辅助能力，不提供投资建议。任何交易和复制交易都可能产生亏损。
{% endhint %}

## 适合谁使用

SeaBond 更适合已经了解 Hyperliquid 基础交易、希望系统化筛选交易地址，并愿意持续复盘风险的用户。如果你只是想无脑跟随某个高收益钱包，SeaBond 并不能消除交易风险。

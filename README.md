# 欢迎使用 SeaBond

SeaBond 是给 Hyperliquid 用户使用的交易发现和跟单工具。你可以在这里找到值得研究的钱包，拆开看它是怎么交易的，把有价值的钱包收藏起来；如果决定跟随，也可以用跟单功能把风险单独管理。

这份文档不会告诉你“应该买什么”。它的作用是帮你看懂 SeaBond 每个页面在解决什么问题，以及在跟随一个钱包前应该检查哪些信息。

## 你可以用 SeaBond 做什么

如果你是第一次打开 SeaBond，可以先按下面这个顺序理解：

1. **先找人**：在聪明钱里筛出表现不错的钱包。
2. **再看细节**：进入钱包详情，确认它到底靠什么赚钱。
3. **先观察**：把钱包收藏起来，用标签记录你的判断。
4. **再决定是否跟单**：确认风险、仓位和资金规模后，再创建跟单。
5. **持续复盘**：在个人中心看自己的账户和跟单表现。

## SeaBond 适合解决的问题

- 我想找一些 Hyperliquid 上表现稳定的钱包，从哪里开始？
- 一个钱包收益很高，但它是不是只靠一笔暴利？
- 它现在有没有重仓？杠杆高不高？适不适合我跟随？
- 我跟单以后，怎么知道收益来自哪里？
- 如果目标钱包风格变了，我应该在哪里发现？

## 先记住三件事

{% hint style="info" %}
SeaBond 更适合“先研究，再跟单”，不适合看到收益高就直接跟。
{% endhint %}

{% hint style="warning" %}
跟单不是自动赚钱工具。目标钱包亏损、市场波动、滑点、延迟和高杠杆都会影响你的账户。
{% endhint %}

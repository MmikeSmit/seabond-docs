# 欢迎使用 SeaBond

SeaBond 是面向 Hyperliquid 用户的聪明钱发现、钱包分析与复制交易平台。用户可以通过 SeaBond 发现高质量交易地址，查看钱包表现，收藏和追踪目标，并基于自己的风险偏好创建复制交易策略。

SeaBond 的目标不是让用户盲目跟单，而是提供更清晰的数据、流程和风险控制工具，帮助用户在理解交易对象后做出决策。

## 核心功能

| 功能 | 说明 |
| --- | --- |
| 聪明钱发现 | 浏览和筛选 Hyperliquid 上表现突出的交易地址。 |
| 钱包分析 | 查看交易者的账户规模、收益表现、持仓和历史交易。 |
| 复制交易 | 根据目标钱包创建复制交易策略，并单独管理复制交易资金。 |
| 个人中心 | 查看账户总览、复制交易账户、统一交易账户和个人资料。 |
| 收藏与标签 | 收藏关注的钱包或市场，并为地址添加便于识别的标签。 |
| 邀请与积分 | 查看邀请关系、奖励状态和积分活动。 |

{% hint style="warning" %}
SeaBond 提供数据分析和交易辅助功能，不构成投资建议。交易和复制交易均存在亏损风险。
{% endhint %}

# 从哪里创建

你可以从两个位置进入跟单：

- 在聪明钱列表中，点击目标钱包的 **跟单**。
- 在钱包详情页中，确认交易表现后点击 **跟单**。

如果从聪明钱或钱包详情进入，带单钱包地址会自动填入。  
如果从个人中心的跟单页面创建，则需要手动输入带单钱包地址。

## 创建流程

1. 打开目标钱包的跟单面板。
2. 确认当前跟随的钱包地址。
3. 选择跟单模式：智能跟单或即时跟单。
4. 输入总投入。
5. 确认可用余额、最低投入和可能产生的激活费。
6. 点击 **跟单** 创建规则。
7. 创建成功后，前往 **个人中心 → 跟单** 查看和管理。

{% hint style="info" %}
跟单创建后由后端服务异步处理。创建成功不代表所有仓位会立刻完成同步，页面数据可能需要短时间更新。
{% endhint %}

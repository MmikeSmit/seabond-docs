# 创建后在哪里看

创建成功后，进入个人中心的跟单页面，可以查看：

- 跟单权益
- 7 日盈亏
- 可用保证金
- 盈亏
- 未实现盈亏
- 当前活跃跟单规则
- 跟单历史

## 管理跟单

在个人中心的跟单页面，可以管理已经创建的跟单规则。

常见操作包括：

| 操作 | 说明 |
| --- | --- |
| 详情 | 查看跟单规则详情。 |
| 关闭 | 停止当前跟单。 |
| 增加保证金 | 为跟单账户增加保证金。 |
| 分享数据 | 分享跟单表现。 |
| 再次跟单 | 基于同一目标钱包再次创建跟单。 |
| 删除 | 删除已关闭或非活跃规则。 |

{% hint style="warning" %}
关闭跟单和删除规则是不同操作。关闭用于停止跟随；删除通常用于清理非活跃记录。
{% endhint %}

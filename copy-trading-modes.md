# 跟单模式

SeaBond 当前提供两种跟单模式。

## 智能跟单

智能跟单是默认模式。它同样按照带单员开仓成本占账户总余额的比例执行，但在同步目标钱包已有持仓时会更谨慎。

如果目标钱包已经有持仓，智能跟单会更谨慎地处理同步。当当前仓位浮动盈亏超过 10% 时，系统会跳过该仓位，等待下一笔交易。

适合：

- 第一次跟随某个钱包的用户
- 希望降低追入已有仓位风险的用户
- 更在意同步质量，而不是立刻跟上所有当前仓位的用户

## 即时跟单

即时跟单会更直接地按照带单员的仓位和资金比例执行订单。它适合希望尽快跟上目标钱包当前交易动作的场景。

适合：

- 已经充分了解目标钱包交易风格的用户
- 希望更快跟随目标交易动作的用户
- 能接受当前市场价格与目标钱包入场价格存在差异的用户

{% hint style="warning" %}
如果目标钱包已有大幅浮盈仓位，即时跟单可能会让你以更差的位置追入。创建前应先查看目标钱包当前持仓。
{% endhint %}

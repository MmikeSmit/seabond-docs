# 复制交易

复制交易是 SeaBond 里风险最高、也最需要认真配置的功能。它可以让你的账户跟随目标钱包交易，但它不会保证你和目标钱包得到一样的结果。

在创建复制交易前，你应该已经看过目标钱包的详情，并且知道自己为什么要复制它。

## 创建前先问自己

不要一看到收益高就复制。先回答这几个问题：

- 这个钱包靠什么赚钱？
- 它亏损时通常怎么处理？
- 它现在有没有重仓？
- 它使用的杠杆我能接受吗？
- 如果我复制后亏损 10% 或 20%，我会不会停止？
- 我是否愿意承担滑点、延迟和执行差异？

如果这些问题答不上来，先收藏观察，不要急着创建策略。

## 复制交易的基本流程

1. 在 Smart Money 找到候选钱包。
2. 进入钱包详情页复盘它的交易方式。
3. 收藏并打标签，观察一段时间。
4. 确认要跟随后，打开复制交易配置。
5. 设置复制资金和策略参数。
6. 创建策略。
7. 在个人中心的 Copies 页面持续复盘。

## 复制资金怎么想

复制资金不是“我最多能投多少”，而是“即使亏掉也不会影响我判断的钱”。

第一次复制建议小额开始。先看执行效果、滑点、目标钱包风格和你的心理承受能力，再决定是否增加资金。

## 当前仓位要不要复制

这是很多用户最容易踩坑的地方。

如果目标钱包已经开仓很久，并且有明显浮盈，你现在复制进去，入场价格和它完全不同。它可以承受回撤，你未必可以。

更适合复制当前仓位的情况：

- 目标钱包刚开仓不久
- 当前价格和目标入场价差距不大
- 仓位不重
- 杠杆不高
- 你能接受价格差异

不建议复制当前仓位的情况：

- 目标钱包已经大幅浮盈
- 仓位很重
- 杠杆很高
- 清算价很近
- 行情已经快速波动过

## 加仓和平仓要不要跟

如果你只跟开仓，不跟加仓和平仓，复制结果可能和目标钱包差很多。

例如目标钱包加仓摊低成本，但你没有跟；或者目标钱包平仓了，但你还留着仓位。时间久了，你的账户和目标钱包会越来越不一样。

所以创建策略时要想清楚：

- 是否跟随开仓
- 是否跟随加仓
- 是否跟随减仓
- 是否跟随平仓
- 是否需要自己额外设置止损

{% hint style="info" %}
复制交易不是一次配置后就不用管。目标钱包风格变化时，你也要重新评估是否继续跟随。
{% endhint %}

## 创建后看什么

复制交易创建后，去个人中心的 Copies 页面查看：

- 当前复制交易权益
- 活跃复制规则
- 已实现和未实现 PnL
- 当前复制持仓
- 历史复制记录

如果连续几次交易结果和目标钱包差异明显，要检查是否因为滑点、资金规模、参数设置或执行延迟造成。

## 什么时候应该停止复制

出现下面情况时，建议重新评估：

- 目标钱包连续亏损
- 目标钱包交易风格明显变化
- 开始频繁高杠杆或重仓
- 复制账户回撤超过你的预算
- 复制价格长期明显差于目标钱包
- 你已经看不懂目标钱包在做什么

{% hint style="warning" %}
复制交易可能产生亏损。SeaBond 只能帮助你管理和观察策略，不能保证收益，也不能替你承担交易风险。
{% endhint %}

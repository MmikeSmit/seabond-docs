# 复制交易

复制交易允许用户根据目标钱包的交易行为创建跟随策略。SeaBond 会将复制交易账户与主钱包直接交易区分开，方便用户单独管理复制资金和跟单表现。

## 创建前检查

在创建复制交易前，建议确认：

- 目标钱包是否有足够长的交易记录
- 收益是否稳定，是否存在极端回撤
- 目标钱包的交易频率是否适合复制
- 自己投入的资金是否可承受损失
- 杠杆、保证金和止损设置是否合理

## 基本流程

1. 在 Smart Money 或钱包详情页选择目标地址。
2. 查看该地址的表现、持仓和历史交易。
3. 点击复制交易入口。
4. 设置复制资金和相关参数。
5. 确认策略并创建。
6. 在个人中心的 Copies 页面持续查看表现。

## 管理复制交易

创建后，用户可以在 Copies 页面查看和管理复制交易。

常见操作包括：

- 查看活跃复制规则
- 停止单个复制策略
- 停止全部复制交易
- 查看历史复制记录

{% hint style="warning" %}
复制交易不是无风险策略。目标钱包亏损、市场剧烈波动、流动性变化或执行延迟都可能导致用户账户亏损。
{% endhint %}

# 复制交易

复制交易允许用户根据目标钱包的交易行为创建跟随策略。SeaBond 会将复制交易账户与主钱包直接交易分开管理，便于用户独立查看复制资金、持仓和历史表现。

## 前置条件

开始复制交易前，请确认：

| 条件 | 说明 |
| --- | --- |
| 已登录 SeaBond | 用于保存复制交易规则和账户配置。 |
| 已连接钱包 | 用于识别用户账户和读取交易相关信息。 |
| 已完成目标钱包分析 | 不建议在未复盘钱包详情的情况下直接复制。 |
| 已准备风险资金 | 复制交易可能亏损，资金规模应在可承受范围内。 |

## 标准流程

1. 在 Smart Money 页面筛选候选钱包。
2. 进入钱包详情页复盘账户表现、持仓和成交历史。
3. 将候选钱包加入收藏并添加标签。
4. 点击复制交易入口。
5. 设置复制资金和规则参数。
6. 确认配置并创建复制交易。
7. 在个人中心的 Copies 页面跟踪表现。

## 参数配置

创建复制交易时，重点关注以下参数。

### 复制资金

复制资金决定该策略最多可使用的资金范围。建议从小额开始，而不是一次性投入全部可用资金。

### 单笔限制

单笔限制用于控制每次跟随开仓的最大风险。对于账户规模较小的用户，单笔限制比总资金更重要。

### 是否复制当前仓位

复制当前仓位表示立即按当前市场价格跟随目标钱包已有持仓。

适用场景：

- 用户明确接受当前价格差异
- 目标仓位仍处于可接受入场区间
- 用户希望立刻对齐目标钱包仓位

不适合：

- 目标钱包已经大幅浮盈
- 当前价格偏离目标钱包入场价太远
- 目标钱包正在高杠杆持仓

### 是否跟随加仓

开启后，当目标钱包对已有仓位继续加仓时，SeaBond 会尝试跟随该行为。

如果关闭，则通常只跟随首次开仓，不跟随后续加仓。

### 是否跟随减仓

开启后，当目标钱包减仓或平仓时，SeaBond 会尝试同步调整。

如果关闭，用户需要自行管理平仓和止盈止损。

{% hint style="info" %}
如果希望复制交易尽量自动化，通常需要同时关注开仓、加仓、减仓和平仓逻辑。
{% endhint %}

## 创建后的管理

创建复制交易后，可以在个人中心的 Copies 页面查看：

- 当前复制交易权益
- 活跃复制规则
- 复制交易 PnL
- 未实现 PnL
- 历史复制记录

常见操作包括：

- 停止单个复制交易
- 停止全部复制交易
- 查看历史记录
- 根据表现决定是否继续跟随

## 复制交易复盘

不要只看复制后的收益，应定期复盘：

- 目标钱包近期是否仍然符合筛选标准
- 复制账户是否出现连续亏损
- 复制价格和目标钱包价格是否长期偏离
- 手续费和滑点是否影响收益
- 是否需要降低复制资金或停止策略

## 风险提醒

{% hint style="warning" %}
复制交易存在市场风险、执行延迟、滑点、保证金不足、强平和目标钱包策略失效等风险。历史表现不代表未来收益，请只使用可承受亏损的资金。
{% endhint %}

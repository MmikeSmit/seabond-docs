# 复制交易

复制交易可以让你用自己的资金跟随一个 Hyperliquid 钱包的交易行为。SeaBond 会为复制交易分配独立的复制账户，并在个人中心里单独展示复制交易权益、规则状态、PnL 和历史记录。

在 SeaBond 中，复制交易不是在主钱包里直接混合执行。主钱包交易和复制交易会分开展示，方便你判断收益和风险来自哪里。

## 从哪里创建复制交易

你可以从两个位置进入复制交易：

- 在 Smart Money 列表中，点击目标钱包的 **Copy**。
- 在钱包详情页中，确认交易表现后点击 **Copy**。

如果从 Smart Money 或钱包详情进入，目标钱包地址会自动填入。  
如果从个人中心的 Copies 页面创建，则需要手动输入 Leader Wallet Address。

## 创建流程

1. 打开目标钱包的复制交易面板。
2. 确认当前复制的钱包地址。
3. 选择复制模式：Smart Copy 或 Instant Copy。
4. 输入 Total Investment。
5. 确认余额、最低投资额和可能产生的激活费用。
6. 点击 **Copy** 创建复制交易。
7. 创建成功后，前往 **Profile → Copies** 查看和管理。

{% hint style="info" %}
复制交易创建后由后端服务异步处理。创建成功不代表所有仓位会立即完成同步，页面数据可能需要短时间更新。
{% endhint %}

## 复制模式

SeaBond 当前提供两种复制模式。

### Smart Copy

Smart Copy 是默认模式。它会根据目标交易者的成本比例进行复制，并在同步已有仓位时加入更谨慎的判断。

如果目标钱包已有仓位，Smart Copy 不会无条件追入所有仓位，而是尽量避免在明显不利的位置强行同步。

适合：

- 希望降低追高已有仓位风险的用户
- 第一次复制某个钱包的用户
- 需要更保守同步逻辑的用户

### Instant Copy

Instant Copy 会更直接地按照目标交易者的仓位和资金比例执行复制。

适合：

- 已经充分了解目标钱包交易风格的用户
- 希望更快跟随目标交易者动作的用户
- 能接受当前市场价格与目标钱包入场价格存在差异的用户

{% hint style="warning" %}
如果目标钱包已有大幅浮盈仓位，Instant Copy 可能会让你以更差的价格追入。创建前应先查看目标钱包当前持仓。
{% endhint %}

## Total Investment

Total Investment 是本次复制交易投入的初始保证金，单位为 USDC。

填写时需要注意：

- 投资金额不能低于系统最低要求。
- 投资金额不能超过当前可用 USDC 余额。
- 页面会显示当前 Balance。
- 可以使用比例滑杆快速选择 25%、50%、75% 或 100%。

{% hint style="info" %}
最低投资额由系统配置决定。页面会在输入金额过低时提示最低要求。
{% endhint %}

## 余额与激活费

如果创建复制交易时需要开通新的复制交易子账户，系统可能会从主钱包收取一次性激活费用。

页面会在需要激活费时显示提示。此费用会影响可投入的最大金额，因此即使钱包里有足够 USDC，也需要预留激活费用。

如果已有可复用的复制交易账户，则通常不需要再次支付该激活费用。

## 无法创建的常见原因

如果 Copy 按钮不可用，通常是以下原因之一：

| 原因 | 说明 |
| --- | --- |
| 未连接钱包 | 需要先连接钱包才能创建复制交易。 |
| 钱包地址无效 | Leader Wallet Address 必须是有效的 EVM 地址。 |
| 投资金额过低 | Total Investment 低于系统最低要求。 |
| 余额不足 | 可用 USDC 不足以覆盖投资金额和可能的激活费。 |
| 目标钱包不可复制 | 部分钱包可能未开启复制交易。 |
| 子账户数量达到上限 | 已达到复制交易子账户上限，需要关闭已有复制策略后再创建。 |

## 创建成功后在哪里看

创建成功后，进入：

```text
/profile?tab=copies
```

在 Copies 页面可以查看：

- Copy Trading Equity
- 7D PNL
- Available Margin
- PnL
- Unrealized PnL
- 当前活跃复制规则
- 历史复制记录

## 管理复制交易

在 Copies 页面，用户可以管理已经创建的复制交易规则。

常见操作包括：

| 操作 | 说明 |
| --- | --- |
| Details | 查看复制规则详情。 |
| Close | 关闭当前复制交易。 |
| Add Margin | 为复制交易账户增加或移除保证金。 |
| Share Stats | 分享复制交易表现。 |
| Copy Again | 基于同一目标钱包再次创建复制交易。 |
| Delete | 删除已关闭或非活跃规则。 |

{% hint style="warning" %}
关闭复制交易和删除规则是不同操作。关闭用于停止跟随；删除通常用于清理非活跃记录。
{% endhint %}

## 查看复制交易表现

不要只看总 PnL。建议同时查看：

- 已实现 PnL
- 未实现 PnL
- 当前复制持仓数量
- Account Value
- MMR
- 已关闭仓位胜率
- 开始时间和结束时间

这些指标能帮助你判断复制交易是否还在可控范围内。

## 风险提示

复制交易会受到市场波动、目标钱包行为、滑点、执行延迟、保证金不足和强平风险影响。

{% hint style="warning" %}
SeaBond 可以帮助你创建和管理复制交易，但不能保证目标钱包盈利，也不能保证你的复制结果与目标钱包完全一致。
{% endhint %}

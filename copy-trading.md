# 跟单

跟单可以让你用自己的资金跟随目标钱包的交易行为。SeaBond 会为跟单分配独立账户，并在个人中心单独展示跟单权益、规则状态、盈亏和历史记录。

在 SeaBond 里，跟单不会和你主钱包里的交易混在一起。主钱包交易和跟单账户会分开展示，方便你判断收益、亏损和风险分别来自哪里。

## 从哪里创建

你可以从两个位置进入跟单：

- 在聪明钱列表中，点击目标钱包的 **跟单**。
- 在钱包详情页中，确认交易表现后点击 **跟单**。

如果从聪明钱或钱包详情进入，带单钱包地址会自动填入。  
如果从个人中心的跟单页面创建，则需要手动输入带单钱包地址。

## 创建流程

1. 打开目标钱包的跟单面板。
2. 确认当前跟随的钱包地址。
3. 选择跟单模式：智能跟单或即时跟单。
4. 输入总投入。
5. 确认可用余额、最低投入和可能产生的激活费。
6. 点击 **跟单** 创建规则。
7. 创建成功后，前往 **个人中心 → 跟单** 查看和管理。

{% hint style="info" %}
跟单创建后由后端服务异步处理。创建成功不代表所有仓位会立刻完成同步，页面数据可能需要短时间更新。
{% endhint %}

## 跟单模式

SeaBond 当前提供两种跟单模式。

### 智能跟单

智能跟单是默认模式。它会根据带单员开仓成本占账户总余额的比例，为你的跟单账户按相同比例执行订单。

如果目标钱包已经有持仓，智能跟单会更谨慎地处理同步。当当前仓位浮动盈亏超过 10% 时，系统会跳过该仓位，等待下一笔交易。

适合：

- 第一次跟随某个钱包的用户
- 希望降低追入已有仓位风险的用户
- 更在意同步质量，而不是立刻跟上所有当前仓位的用户

### 即时跟单

即时跟单会更直接地按照带单员的仓位和资金比例执行订单。

适合：

- 已经充分了解目标钱包交易风格的用户
- 希望更快跟随目标交易动作的用户
- 能接受当前市场价格与目标钱包入场价格存在差异的用户

{% hint style="warning" %}
如果目标钱包已有大幅浮盈仓位，即时跟单可能会让你以更差的位置追入。创建前应先查看目标钱包当前持仓。
{% endhint %}

## 总投入

总投入是本次跟单投入的初始保证金，单位为 USDC。

填写时需要注意：

- 金额不能低于系统最低要求。
- 金额不能超过当前可用 USDC 余额。
- 页面会显示当前余额。
- 可以使用比例滑杆快速选择 25%、50%、75% 或 100%。

{% hint style="info" %}
最低投入由系统配置决定。页面会在输入金额过低时提示最低要求。
{% endhint %}

## 余额与激活费

如果创建跟单时需要开通新的跟单子账户，系统会从主钱包额外收取 1 USDC 激活费。

页面会在需要激活费时显示提示。该费用会影响可投入的最大金额，因此即使钱包里有足够 USDC，也需要预留激活费。

如果已有可复用的跟单账户，通常不需要再次支付该费用。

## 无法创建的常见原因

如果 **跟单** 按钮不可用，通常是以下原因之一：

| 原因 | 说明 |
| --- | --- |
| 未连接钱包 | 需要先连接钱包才能创建跟单。 |
| 钱包地址无效 | 带单钱包地址必须是有效的 EVM 地址。 |
| 投入金额过低 | 总投入低于系统最低要求。 |
| 余额不足 | 可用 USDC 不足以覆盖投入金额和可能的激活费。 |
| 目标钱包不可跟单 | 部分钱包可能已关闭跟单。 |
| 子账户数量达到上限 | 已达到跟单子账户上限，需要先结束一笔现有跟单再创建新的跟单。 |

## 创建后在哪里看

创建成功后，进入个人中心的跟单页面，可以查看：

- 跟单权益
- 7 日盈亏
- 可用保证金
- 盈亏
- 未实现盈亏
- 当前活跃跟单规则
- 跟单历史

## 管理跟单

在个人中心的跟单页面，可以管理已经创建的跟单规则。

常见操作包括：

| 操作 | 说明 |
| --- | --- |
| 详情 | 查看跟单规则详情。 |
| 关闭 | 停止当前跟单。 |
| 增加保证金 | 为跟单账户增加保证金。 |
| 分享数据 | 分享跟单表现。 |
| 再次跟单 | 基于同一目标钱包再次创建跟单。 |
| 删除 | 删除已关闭或非活跃规则。 |

{% hint style="warning" %}
关闭跟单和删除规则是不同操作。关闭用于停止跟随；删除通常用于清理非活跃记录。
{% endhint %}

## 查看跟单表现

不要只看总盈亏。建议同时查看：

- 已实现盈亏
- 未实现盈亏
- 当前跟单持仓数量
- 账户价值
- 维持保证金率
- 已关闭仓位胜率
- 开始时间和结束时间

这些指标能帮助你判断跟单是否仍在可控范围内。

## 风险提示

跟单会受到市场波动、目标钱包行为、滑点、执行延迟、保证金不足和强平风险影响。

{% hint style="warning" %}
SeaBond 可以帮助你创建和管理跟单，但不能保证目标钱包盈利，也不能保证你的跟单结果与目标钱包完全一致。
{% endhint %}

# 收藏与标签

收藏与标签用于帮助用户管理关注对象。用户可以收藏钱包或市场，并为地址添加标签，便于后续筛选和识别。

## 收藏钱包

当用户发现值得持续观察的钱包时，可以将其加入收藏。收藏后，用户可以更快回到该钱包并查看最新表现。

## 地址标签

标签适合用于记录用户自己的判断，例如：

- 高频交易
- 稳健型
- 重仓风险
- 观察中
- 已复制

{% hint style="info" %}
标签是用户自己的管理工具，不代表平台官方评级。
{% endhint %}

## 使用建议

建议将收藏和标签作为研究流程的一部分，而不是直接作为复制交易依据。对重要钱包应持续观察其仓位变化和风险表现。

# 收藏与标签

收藏和标签用于管理你正在观察的钱包。它们不是结论，而是研究过程的一部分。

当你在聪明钱或钱包详情里看到一个值得关注的钱包，可以先收藏，再用标签记录你关注它的原因。

## 为什么需要标签

同一个钱包今天看起来不错，过几天可能已经不适合跟随。标签可以帮你快速回忆当时的判断。

常见标签可以包括：

- 观察中
- 主流币波段
- 高频交易
- 高杠杆
- 回撤偏大
- 可考虑跟单
- 已跟单
- 已排除

标签不需要复杂，关键是让你下次回来时能快速理解这个钱包的状态。

## 推荐用法

1. 在聪明钱里筛出候选钱包。
2. 进入详情页查看持仓和历史交易。
3. 对值得继续看的钱包添加收藏。
4. 用标签记录关注原因和风险点。
5. 观察一段时间后再决定是否跟单。

## 使用边界

收藏不等于已经做出跟单判断。它更适合用来保存候选钱包，方便后续继续查看表现变化。

{% hint style="warning" %}
收藏和标签适合作为研究流程的一部分。对重要钱包，可以持续观察其仓位变化和风险表现。
{% endhint %}

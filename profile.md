# 个人中心

个人中心是你查看自己账户状态的地方。找钱包和创建复制交易之后，最终都要回到这里看结果：钱在哪里、复制交易赚亏多少、自己的主账户有没有风险。

## 什么时候需要看个人中心

你通常会在这些时候打开个人中心：

- 创建复制交易后，想看复制账户表现
- 想确认自己的总资产变化
- 想查看主钱包持仓和挂单
- 想看充值、提现或交易历史
- 想修改头像、昵称或个人简介

## Overview：先看整体

Overview 是总览页，适合先回答一个问题：我现在账户整体怎么样？

这里会把主钱包和复制交易账户放在一起看，适合查看：

- 当前估算总资产
- 7 日 PnL
- 资产分布
- 充值和提现记录
- 个人资料入口

如果你只想快速知道账户整体变化，先看 Overview。

## Copies：看复制交易

Copies 只看复制交易账户，不混入你自己主钱包的交易。

你可以在这里确认：

- 复制交易权益是多少
- 哪些复制规则还在运行
- 当前复制交易赚亏多少
- 是否有未实现亏损
- 历史复制交易表现如何

{% hint style="warning" %}
复制交易创建、关闭或调整后，数据可能不会立刻完全同步。短时间内看到的状态可能还在更新。
{% endhint %}

## Perpetuals：看自己的交易账户

Perpetuals 看的是你自己的主钱包统一交易账户。

这里适合查看：

- 当前持仓
- 现货资产
- 成交历史
- 当前挂单
- 资金费率记录
- 历史订单

如果你想区分“自己交易赚亏”和“复制交易赚亏”，就分别看 Perpetuals 和 Copies。

## My Profile：管理个人资料

点击 **My Profile** 可以修改个人信息和查看账户等级。

可以管理：

- 头像
- 展示名称
- UID
- VIP 等级
- 个人简介
- 手续费信息

## 数据怎么看才不混乱

记住这三个口径：

- **Overview**：主钱包 + 复制交易账户
- **Copies**：只看复制交易账户
- **Perpetuals**：只看自己的主钱包交易账户

这样你就能判断收益到底来自自己的交易，还是来自复制交易。

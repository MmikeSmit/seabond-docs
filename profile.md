# 个人中心

个人中心用于查看与管理当前账户。这里展示账户资料、整体资产表现、复制交易账户和用户自己的统一交易账户。

## 进入页面

| 页面 | 地址 |
| --- | --- |
| 个人中心 | `/profile` |
| 复制交易账户 | `/profile?tab=copies` |
| 统一交易账户 | `/profile?tab=perpetuals` |

{% hint style="info" %}
个人中心需要登录后访问。未登录用户会被引导至 Smart Money 页面。
{% endhint %}

## 页面组成

| 标签 | 用途 |
| --- | --- |
| Overview | 查看主钱包与复制交易账户合并后的整体表现。 |
| Copies | 查看复制交易权益、活跃复制规则和复制交易历史。 |
| Perpetuals | 查看用户自己主钱包的持仓、资产、挂单和交易记录。 |

## Overview

Overview 是账户总览页面，适合快速确认当前账户表现。

主要内容包括：

- 账户估算总值
- 7 日 PnL
- 资产分布
- 充值与提现记录
- 个人资料入口

## Copies

Copies 用于查看复制交易账户。该页面只展示复制交易相关资金和规则，不与用户自己的主钱包交易混在一起。

用户可以在这里：

- 查看复制交易权益
- 查看当前活跃复制规则
- 创建新的复制交易
- 停止全部复制交易
- 查看历史复制记录

{% hint style="warning" %}
复制交易状态可能存在短暂延迟。创建、关闭或调整策略后，页面数据可能需要稍等片刻才会更新。
{% endhint %}

## Perpetuals

Perpetuals 展示用户自己的统一交易账户，适合查看直接交易情况。

主要内容包括：

- 当前持仓
- 现货资产
- 成交历史
- 当前挂单
- 资金费率记录
- 历史订单

## 个人资料

点击页面中的 **My Profile** 可以进入个人资料设置。

可管理内容包括：

- 头像
- 展示名称
- UID
- VIP 等级
- 个人简介
- 当前手续费信息

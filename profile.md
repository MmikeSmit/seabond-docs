# 个人中心

个人中心是查看账户状态和跟单结果的地方。你在聪明钱里找到钱包、在详情页判断是否值得跟随，最后都要回到这里看结果：资金在哪里，跟单赚亏多少，自己的主账户有没有风险。

## 什么时候需要看个人中心

你通常会在这些时候打开个人中心：

- 创建跟单后，查看跟单账户表现
- 确认自己的总资产变化
- 查看主钱包持仓、挂单和历史订单
- 查看充值、提现或交易历史
- 修改头像、昵称或个人简介

## 总览：先看整体

总览适合先回答一个问题：我现在账户整体怎么样？

这里会把主钱包和跟单账户放在一起看，适合查看：

- 当前估算总资产
- 7 日盈亏
- 资产分布
- 充值和提现记录
- 个人资料入口

如果你只想快速知道账户整体变化，先看总览。

## 跟单：看跟单账户

跟单页面只看跟单账户，不混入你自己主钱包的交易。

你可以在这里确认：

- 跟单权益是多少
- 哪些跟单规则还在运行
- 当前跟单赚亏多少
- 是否有未实现盈亏
- 历史跟单表现如何

{% hint style="warning" %}
跟单创建、关闭或调整后，数据可能不会立刻完全同步。短时间内看到的状态可能还在更新。
{% endhint %}

## 统一交易：看自己的交易账户

统一交易看的是你自己的主钱包交易账户。

这里适合查看：

- 当前持仓
- 现货资产
- 成交历史
- 当前挂单
- 资金费率记录
- 历史订单

如果你想区分“自己交易赚亏”和“跟单赚亏”，就分别看统一交易和跟单。

## 我的资料：管理个人信息

点击个人资料入口可以修改个人信息和查看账户等级。

可以管理：

- 头像
- 显示名称
- UID
- VIP 等级
- 个人简介
- 手续费信息

## 怎么看才不混

记住这三个入口：

- **总览**：主钱包 + 跟单账户
- **跟单**：只看跟单账户
- **统一交易**：只看自己的主钱包交易账户

这样你就能判断收益到底来自自己的交易，还是来自跟单。

# 快速开始

本页介绍从首次进入 SeaBond 到完成第一轮钱包筛选的基础流程。

## 前置准备

在使用 SeaBond 前，建议准备：

| 项目 | 说明 |
| --- | --- |
| 登录账户 | 用于保存收藏、标签、邀请关系和个人设置。 |
| 钱包 | 用于查看资产、交易记录和进行后续交易操作。 |
| USDC 资金 | 如需复制交易，需要准备可承担风险的资金。 |
| 风险预算 | 先确定最大可承受亏损，不要先看收益再决定投入。 |

## 第一步：登录

进入 SeaBond 后先完成登录。登录后，平台会同步用户的收藏、标签、复制交易规则、积分和个人资料。

如果未登录，部分页面只能浏览公开数据，无法保存个人操作。

## 第二步：连接钱包

连接钱包后，SeaBond 才能读取与你相关的账户信息，包括：

- 主钱包资产
- 当前持仓
- 交易历史
- 复制交易账户
- 可用保证金

{% hint style="info" %}
连接钱包不等于创建复制交易。复制交易需要用户单独确认并配置参数。
{% endhint %}

## 第三步：进入聪明钱发现

在 Smart Money 页面先做第一轮筛选。建议不要一开始就点进每个详情页，而是先用筛选条件排除明显不适合的钱包。

常见第一轮筛选条件：

- 时间周期选择 30D 或 All
- 胜率高于自己的最低要求
- 账户规模不能过小
- 已实现利润不能只靠单笔拉升
- 交易次数不能过少，也不宜过度高频
- 回撤不能明显大于收益

## 第四步：进入钱包详情复盘

通过第一轮筛选后，再进入钱包详情页查看：

- 收益曲线是否平滑
- 当前持仓是否过度集中
- 杠杆是否过高
- 平均持仓时间是否符合你的跟单偏好
- 历史交易是否存在频繁大亏

## 第五步：收藏和打标签

不要马上复制交易。建议先把候选钱包加入收藏，并用标签记录判断，例如：

- 观察中
- 主流币波段
- 高杠杆
- 回撤偏大
- 已排除
- 可考虑复制

## 第六步：创建复制交易

只有在完成筛选和复盘后，才进入复制交易配置。创建前至少确认：

- 复制资金规模
- 单笔最大跟随金额
- 是否跟随加仓和减仓
- 是否复制当前仓位
- 是否设置止盈止损

{% hint style="warning" %}
复制交易不是筛到一个高收益钱包后就结束。后续仍需要定期复盘目标钱包表现，并及时淘汰不符合预期的钱包。
{% endhint %}

# 快速开始

本页介绍使用 SeaBond 的基本流程。

## 1. 登录账户

进入 SeaBond 后，用户需要先完成登录。登录用于识别当前账户，并同步个人设置、收藏、复制交易规则和邀请信息。

## 2. 连接钱包

登录后，连接用于交易和查看资产的钱包。连接钱包后，SeaBond 才能展示账户余额、持仓、交易记录和复制交易相关信息。

## 3. 浏览聪明钱

进入 Smart Money 页面，查看平台整理的钱包列表。用户可以根据收益表现、账户规模、持仓方向和交易行为筛选目标地址。

## 4. 查看钱包详情

打开目标钱包详情页，重点查看：

- 账户规模
- 历史收益
- 当前持仓
- 交易频率
- 风险特征

## 5. 创建复制交易

如果决定复制某个地址，可以在钱包详情页或复制交易入口创建策略。

{% hint style="warning" %}
创建复制交易前，请确认复制资金、杠杆、止损和最大可承受亏损。不要只根据短期收益决定是否跟单。
{% endhint %}

## 6. 查看个人中心

复制交易创建后，可在个人中心查看整体账户、复制交易账户和历史记录。

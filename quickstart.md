# 快速开始

这一页只讲第一次使用 SeaBond 时最应该做什么。目标不是马上创建复制交易，而是先跑通一次完整的“发现 - 分析 - 收藏 - 决策”流程。

## 第一次进来先做什么

### 1. 登录

登录后，SeaBond 才能保存你的收藏、标签、邀请关系、个人资料和复制交易规则。

如果你只是浏览公开钱包数据，可以先不急着操作；但只要你想收藏、标记或创建复制交易，就需要登录。

### 2. 连接钱包

连接钱包后，你可以看到自己的账户、持仓、交易历史和复制交易相关数据。

{% hint style="info" %}
连接钱包只是让 SeaBond 读取和展示你的账户信息，并不代表已经开始复制交易。
{% endhint %}

### 3. 进入 Smart Money

Smart Money 是找交易钱包的入口。第一次使用时，不建议一上来就找“收益最高”的钱包，而是先排除明显不适合复制的钱包。

可以先看这些条件：

- 账户规模是否太小
- 收益是不是只靠一两笔大单
- 交易次数是否太少
- 回撤是否过大
- 杠杆是否明显偏高
- 近期表现和长期表现是否一致

### 4. 打开钱包详情

筛出几个候选钱包后，再进入详情页。你要看的不是“这个钱包赚了多少”，而是“它怎么赚的”。

重点看：

- 当前是否有重仓
- 是否经常高杠杆
- 盈利是否稳定
- 亏损时会不会一直加仓
- 平仓是否有纪律
- 最近策略是否已经变了

### 5. 先收藏，不急着复制

如果一个钱包看起来不错，先收藏并打标签。比如：

- 观察中
- 主流币波段
- 高杠杆
- 回撤偏大
- 可考虑复制
- 已排除

这样你下次回来不会重新分析一遍。

### 6. 再考虑复制交易

只有当你理解这个钱包的交易方式，并且知道自己最多能亏多少时，再创建复制交易。

创建前至少想清楚：

- 我准备用多少钱复制？
- 如果连续亏损，我什么时候停？
- 我要不要复制它当前已有仓位？
- 它加仓时，我是否也跟着加？
- 它平仓时，我是否自动跟着平？

{% hint style="warning" %}
第一次使用 SeaBond，不建议直接大额复制。先小资金测试，观察执行效果和目标钱包风格是否稳定。
{% endhint %}

# 聪明钱发现

聪明钱是你在 SeaBond 里找交易钱包的地方。它不是把“最赚钱的钱包”直接摆在你面前，而是让你更快找到值得继续研究的钱包。

一个钱包收益高，不代表它适合跟随。你真正要找的是：风格你能理解、风险你能承受、表现不是偶然的钱包。

## 先不要急着跟单

很多用户第一次看聪明钱列表，会自然被高收益吸引。但高收益钱包里通常混着几类风险：

- 账户很小，一笔交易就把收益率拉得很高
- 只靠某个币种的一次暴利
- 杠杆极高，下一次可能大幅回撤
- 交易频率太高，你跟随时可能跟不上
- 近期表现很好，但长期表现普通

所以聪明钱的正确用法是：先筛选，再复盘，最后才决定是否跟单。

## 怎么筛第一批候选钱包

你可以先用比较硬的条件缩小范围。

### 看账户规模

账户太小的钱包容易出现数据失真。比如一个小账户刚好抓住一笔大行情，收益率会非常好看，但不一定代表策略稳定。

### 看收益来源

不要只看总盈亏。你需要判断收益是不是来自持续交易，还是只靠一两笔大单。

如果一个钱包大部分收益都来自单笔交易，它可以收藏观察，但不适合马上跟单。

### 看交易次数

交易次数太少，样本不够；交易次数太多，可能是高频或短线策略。高频策略跟随时更容易受滑点、延迟和手续费影响。

### 看回撤

回撤比收益更重要。一个钱包如果赚得多但回撤也很深，你跟单时未必能扛住中间那段亏损。

### 看近期和长期是否一致

近期突然爆发的钱包要谨慎。更值得观察的是：近期表现不错，同时长期也没有明显失控。

## 看到一个钱包后怎么判断

可以按这个顺序看：

1. 它是不是长期稳定，而不是短期暴利？
2. 它有没有明显重仓或高杠杆习惯？
3. 它主要交易哪些币？你是否能接受这些币的波动？
4. 它亏损时会不会继续加仓？
5. 它平仓是否有规律？
6. 它现在的持仓是否已经不适合追进去？

{% hint style="info" %}
聪明钱列表适合做第一轮筛选。真正决定是否跟单前，一定要进入钱包详情页继续看。
{% endhint %}

## 推荐的工作流

1. 用筛选条件找出 10 到 20 个候选钱包。
2. 打开详情页，排除明显高风险的钱包。
3. 把剩下的钱包加入收藏。
4. 给每个钱包打标签，记录你为什么关注它。
5. 观察一段时间后，再决定是否跟单。

## 什么时候应该排除一个钱包

如果出现下面情况，建议直接排除或只放入观察：

- 账户规模过小但收益率极高
- 收益曲线主要靠单笔暴利拉升
- 当前持仓已经大幅浮盈
- 经常满仓或高杠杆交易
- 回撤明显超过你的承受范围
- 最近交易风格和过去完全不同

{% hint style="warning" %}
SeaBond 可以帮你更快发现钱包，但不能替你判断风险。跟单前请先确认这个钱包的交易方式是否真的适合你。
{% endhint %}

# 聪明钱发现

Smart Money 页面用于从 Hyperliquid 钱包中筛选值得进一步研究的交易地址。这个页面的重点不是“找到收益最高的钱包”，而是快速排除噪音，建立候选池。

## 使用目标

用户可以通过 Smart Money 完成：

- 按收益、胜率、账户规模和交易行为筛选钱包
- 查看不同钱包的核心指标
- 识别高收益但高风险的钱包
- 将候选钱包加入收藏
- 进入钱包详情页做进一步复盘
- 为复制交易建立备选池

## 筛选思路

建议按四步完成筛选。

### Step 1：先用硬条件排除噪音

第一轮筛选的目标是减少样本数量，而不是直接选出要复制的钱包。

可参考以下筛选维度：

| 维度 | 关注点 | 说明 |
| --- | --- | --- |
| 时间周期 | 30D / All | 30D 适合观察近期表现，All 适合判断长期稳定性。 |
| 账户规模 | 不宜过小 | 账户过小的钱包容易因单笔交易造成指标失真。 |
| 已实现 PnL | 是否持续盈利 | 不要只看总收益，还要看收益来源。 |
| 胜率 | 是否稳定 | 胜率高但盈亏比差，也可能不适合复制。 |
| 交易次数 | 样本是否足够 | 交易次数太少难以判断稳定性，过高可能是高频策略。 |
| ROI | 资金效率 | 结合账户规模一起看，避免被小资金高波动误导。 |

{% hint style="info" %}
第一轮筛选建议使用明确阈值。例如账户规模、最低 PnL、最低胜率和交易次数范围。这样可以显著降低后续分析成本。
{% endhint %}

### Step 2：看 PnL 曲线

筛选后先观察收益曲线，而不是立刻复制。

优先排除：

- 曲线大幅震荡、回撤很深的钱包
- 长时间亏损后靠单笔暴利拉升的钱包
- 近期收益突然异常放大的钱包
- 收益曲线与账户规模不匹配的钱包

更值得继续观察的钱包通常具备：

- 收益曲线相对连续
- 回撤可控
- 盈利不是单笔交易贡献
- 不依赖极端高杠杆

### Step 3：查看交易统计

进入详情前，可以先通过交易统计判断钱包风格。

重点关注：

| 指标 | 判断方式 |
| --- | --- |
| 交易币种 | 主流币更多通常波动较低，山寨币更多通常波动更高。 |
| 最佳交易收益 | 如果总收益主要来自少数暴利单，需要降低评分。 |
| 多空分布 | 判断钱包是否长期单边押注。 |
| 平均杠杆 | 杠杆越高，复制时对滑点和回撤越敏感。 |
| 持仓时长 | 高频、短线、波段和长线适合不同复制方式。 |

### Step 4：进入钱包详情做最终判断

详情页用于确认该钱包是否真的适合加入候选池。不要只看一个指标，应同时看持仓、成交历史和资金变化。

建议重点复盘：

- 当前是否有重仓持仓
- 是否经常浮亏加仓
- 是否频繁使用高杠杆
- 是否在极端行情中出现大额亏损
- 平仓是否有纪律
- 近期策略是否和历史表现一致

## 维护候选池

聪明钱筛选不是一次性工作。建议定期维护候选池：

- 每周复盘收藏钱包
- 删除表现明显变差的钱包
- 给已排除的钱包加标签，避免重复分析
- 将风格不同的钱包分组管理
- 对已复制的钱包保持单独标记

{% hint style="warning" %}
高收益钱包不一定适合复制。账户规模、交易频率、杠杆、持仓集中度和回撤控制，都可能比短期收益更重要。
{% endhint %}

# 聪明钱发现

Smart Money 是 SeaBond 的核心发现页，用于帮助用户寻找值得进一步研究的 Hyperliquid 交易地址。

## 适用场景

用户可以使用 Smart Money：

- 发现表现突出的交易地址
- 比较不同钱包的收益和风险特征
- 查看目标钱包的当前持仓
- 收藏需要持续关注的钱包
- 作为复制交易前的研究入口

## 如何判断一个钱包

建议不要只看单一收益指标，而是综合观察：

| 维度 | 关注点 |
| --- | --- |
| 收益表现 | 是否有稳定收益，是否只依赖单次大额盈利。 |
| 回撤风险 | 盈亏波动是否过大，是否频繁重仓。 |
| 持仓结构 | 是否集中在单一币种或方向。 |
| 交易频率 | 是否适合复制交易，是否过于高频。 |
| 账户规模 | 钱包资金体量是否与自己的复制资金匹配。 |

## 收藏与追踪

如果某个地址值得持续观察，可以将其加入收藏。收藏后的地址可在 Favorites 页面集中查看，也可以结合标签进行分类。

{% hint style="info" %}
Smart Money 用于辅助研究，不代表平台对某个钱包的收益作出保证。
{% endhint %}
